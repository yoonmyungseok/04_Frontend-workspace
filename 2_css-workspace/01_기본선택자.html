CSS 기본 선택자

선택자란?

특정 html 요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 "스타일"과 기능을 적용시킬 수 있다

1. 모든 (전체) 선택자 (*)

모든 요소들을 다 선택하고자 할 때 사용 

        [표현법]
        *{
            스타일속성명: 값;
            스타일속성명: 값;
            ...
        }

2. 태그 선택자(태그명)

해당 이 문서 내에 해당 같은 태그들을 모두 선택할 때 사용됨 

        [표현법]
        태그명 {
            스타일속성명: 값;
            스타일속성명: 값;
            ...
        }

3. 아이디 선택자 (#아이디속성값)

해당 이 문서 내에 특정 html 요소 하나만을 선택하고자 할 때 사용
단, 해당 그 요소 id 속성을 이용할 때에는 고유한 id 값을 부여해야 한다.(중복 불가) 

        [표현법]
        #아이디속성값{
            스타일속성명: 값;
            스타일속성명: 값;
            ...
        }
