글꼴 관련 스타일 속성 (font~)

font-family : 글꼴을 지정할 때 사용

        [ 표현법 ]

        선택자 { font-family : 글꼴명; }

font-size : 글꼴의 크기를 변경할 때 사용

        [ 표현법 ]

        선택자 { font-size : 크기(px/em/%); }
  • 고정크기 px 테스트
  • 가변크기 em 테스트
  • 가변크기 % 테스트

font-weight : 글꼴의 굵기를 표현할 때 사용

        [ 표현법 ]

        선택자 { font-weight : normal/bold/bolder/lighter/100/400/900; }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 텍스트문구를 작은 대문자로 변경할 때 사용

        [ 표현법 ]

        선택자 { font-variant : small-caps; }

font-style : 텍스트문구를 기울이고자 할 때 사용

        [ 표현법 ]

        선택자 { font-style : normal/italic/oblique; }
  • italic : 기울임 글꼴
  • oblique : 원래 글자를 기울인 셈
