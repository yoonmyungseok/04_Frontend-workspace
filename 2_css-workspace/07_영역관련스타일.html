여백 및 간격 관련 스타일

기준 div

기준 div

padding

내용물영역(content)과 테두리영역(border) 사이의 거리(여백)을 조정하는 스타일 속성
(패딩을 따로 지정하지 않으면 기본값으로 0px이 잡힌다)

컨텐츠 영역
padding: 100px;

컨텐츠 영역
padding-top : 50px; padding-left: 10px;

컨텐츠 영역
padding: 50px 20px 10px 0px;

margin

타 요소들 간의 간격을 조정할 때 사용 (바깥 여백, 실제 요소의 범위가 아님)

컨텐츠 영역
margin: 50px;

컨텐츠 영역
margin-top:50px; margin-left:20px;
컨텐츠 영역
margin: auto;
