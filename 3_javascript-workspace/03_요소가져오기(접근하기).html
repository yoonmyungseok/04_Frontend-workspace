HTML 요소에 접근하기 (해당 요소객체를 코드상으로 가져오기)

  1. 아이디값으로 가져오기: document.getElementById('아이디속성값'); =>요소 한개만
  2. 태그명으로 가져오기: document.getElementsByTagName('태그명'); =>요소 여러개가 배열에 담겨서
  3. name속성명으로 가져오기: document.getElementsByName('name속성값'); =>요소 여러개가 배열에 담겨서
  4. 클래스명으로 가져오기: document.getElementsByClassName('class속성값'); =>요소 여러개가 배열에 담겨서
