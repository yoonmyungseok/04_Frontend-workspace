변수와 자료형

* 변수 선언 시 var 뿐만 아니라 let, const도 사용 가능

var, let, const의 차이점

1) 변수 선언 방식

2) 스코프(변수 사용 가능 영역)


* 변수 선언 위치에 따른 전역변수/지역변수

        [표현법]

        <script>
            var 변수명; //전역변수(global)

            변수명; //전역변수(global)

            function 함수명(){
                var 변수명; //지역변수(local)
                변수명; //전역변수(global)
            }
        </script>

* 자료형

자바스크립트에서는 변수 선언 시 자료형을 별도로 지정하지 않음 (단, 자료형의 개념이 없는건 아님)
=>변수에 대입되는 값(리터럴)에 따라서 알아서 자료형이 자동으로 결정됨

자바스크립트의 자료형

  • string (문자열)
  • number (숫자)
  • boolean (논리형)
  • object (객체형)
  • function (함수)
  • undefined (초기화가 안된 변수)
