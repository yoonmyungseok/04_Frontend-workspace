함수 (function)

* 선언적 함수(함수명이 있을 경우)

        [표현법]
        function 함수명(매개변수, 매개변수, ...){

            해당 함수 호출 시 실행할 소스코드들;

            return 결과값;
        }
        -매개변수가 없는 경우 매개변수를 생략 가능
        -반환할 결과값이 없는 경우 return 결과값; 구문 생략 가능

* 익명 함수(함수명이 없을 경우)

        [표현법]
        변수=function (매개변수명, 매개변수명, ...){
            소스코드;
        };
        
        -특정 변수나 속성에 대입되는 함수 제시시 주로 사용됨
        (주로 이벤트가 발생되었을 때 실행할 코드를 작성할 용도로 쓰임==이벤트핸들러함수)
