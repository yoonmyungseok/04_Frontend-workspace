이벤트 핸들러 (이벤트 발생 시 실행될 function) 연결 방법 종류

* 방법1. 이벤트 메소드를 통한 연결

        [표현법]
        $("선택자").이벤트메소드(function(){
            해당 요소에 해당 이벤트 발생 시 실행할 내용;
        })

클릭해보세요



* 방법2. on 메소드를 이용한 방법

        [표현법]
        $("선택자").on('이벤트명', function(){
            해당 요소에 해당 이벤트 발생 시 실행할 내용;
        });

마우스 클릭 및 올려보세요



* 방법3. on 메소드를 이용하는 방법2

        [표현법]
        $('타겟의부모선택자').on('이벤트명', '하위요소선택자==타겟', function(){
            선택된 타겟의 부모선택자 안에 존재하는 
            해당 하위요소의 해당 이벤트가 발생했을 때 실행할 내용;
        });

h4를 클릭해보세요

h5를 클릭해보세요

* 주의사항

단, 동적으로 만들어진 요소에도 동일한 이벤트를 적용시키고자 한다면 반드시 3번 방법으로만 진행해야함

기존에 존재하는 요소

* 참고 - 일회성 이벤트

이벤트를 단 한 번만 연결할 때 사용
[표현법]
$('선택자').one('이벤트명',function(){
이벤트 발생 시 실행할 내용;
});

단 한 번만 실행할 것

* 참고 - 키보드 관련 이벤트

-keydown / keypress: 키가 눌려질 때 발생하는 이벤트들
keydown: 키보드에 모든 키가 눌려질 때 발생됨
keypress: 키보드의 펑션키, 기능키, 한글 제외한 나머지 키들이 눌렸을 때 발생됨
-keyup: 키가 떼어질 때 발생하는 이벤트
